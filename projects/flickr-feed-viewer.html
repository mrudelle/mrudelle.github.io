Flickr Feed Viewer

Instantly navigate through the Flickr gallery
Short opens-source project to get familiar with angular and Sass

April/May 2014 - Web Development

Angular is a powerful tool to add interactivity to your website. I really liked the one-page-app principle that is easy to setup and the ability to interact easily with the data.

The idea was to build a web application to have a quick access to the public database of images on flickr. For this the title of the page can be edited and while the user is typing, the app fetches the coresponding images on the fly and provide a quick visual feedback. Then tag list allows the user to quickly navigate from a tag to another.

The project is entirely open-source and is available frome this github repository

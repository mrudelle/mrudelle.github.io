Artworks Visualization

A new way to visualize artworks
Summer internship, research project and part time position at Artmyn

2014-2016 - Web Development - Artmyn

Artmyn is a startup from LCAV laboratory at EPFL. The startup is doing researches related to the visualization of artwork in the more realistic manner possible, notably through relighting. Since an internship in summer 2014 I am now working part-time as lead developer for the visualization web app.
The web app uses WebGL to render the artwork in 3D and with high resolution. Among others, it implements the following key features:

  • Virtual Texturing: this is an 'infinite zoom' like feature, similar to what Google Maps proposes. It streams small parts of the texture as a client visit the artworks.
  • Adaptive Mesh: allows to have a very high geometry precision when zooming on the artwork and keeping the vertex count low at the same time.
  • Guided Tour: animates the web app like if it was playing a video with sound. Except the animation is rendered live.
  • Relighting: allows the user to change the light conditions (direction, temperature, specularity ...) while the artwork reacts realistically.
The app is provided with an intuitive viewer interface for desktop and mobile devices but other interfaces like Leap Mothion are being tested!

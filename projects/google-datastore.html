Datastore Interactive Tutorial

An interractive tutorial to learn and
experiment in real-time with Cloud Datastore

Summer 2015 - Web Development - Google, Mountain View, CA

In an effort to get more developers discovering and considering Datastore for web app development, my project at Google was to build a framework for interactive tutorial and apply it to Datastore's RESTful API.
I built a framework for exercising the user provided content leveraging a module to assess the execution of the user's solution and provide feedback on the exercise. The UI module allows the user to travel tutorial steps and edit the solution of the exercise.
